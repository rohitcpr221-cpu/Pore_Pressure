# ⛽ Pore Pressure Prediction using Machine Learning

## 📌 Project Overview
This project predicts pore pressure in the Krishna-Godavari Basin using Machine Learning techniques. The model is trained on well log data collected from 8 different wells to estimate subsurface pore pressure accurately.

## 🎯 Objectives
- Predict pore pressure from well log data.
- Perform data preprocessing and analysis.
- Train and compare multiple ML models.
- Select the best-performing model.

## 🛠 Technologies Used
- Python
- Pandas
- NumPy

- Google Colab

## 🤖 Models Used
- Linear Regression
- Random Forest Regressor
- Gradient Boosting Regressor

## 📊 Evaluation Metrics
- MAE (Mean Absolute Error)
- RMSE (Root Mean Squared Error)
- R² Score

## 📈 Project Workflow
Data Collection → Data Preprocessing → EDA → Feature Selection → Model Training → Model Evaluation → Best Model Selection

## 🚀 Key Features

✔ Compares multiple ML models
